1. Qu’est-ce que Lucene ? Est-ce destiné à Java uniquement ?

Apache Lucene est un moteur de recherche de texte écrit completement en Java. C'est une technologie écrite entièrement en Java et qui est utilisable avec n'importe quelle application et sur n'importe quelle plateforme.

C'est un projet Open Source.

Il est destiné pour Java mais d'autres versions existe, comme en .NET ou en C.
